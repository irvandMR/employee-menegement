## Recommended

node v18.13.0

## Installing

Berikut cara menjalankan aplikasi :

##### Clone Repo

```bash
$ git clone https://github.com/irvandMR/employee-menegement.git
```

##### Install Package

```bash
$ npm i
```

##### Start Project

```bash
$ npm start
```
